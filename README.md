# Headlamp plugins

This repository publishes Headlamp plugins maintained by Stian Frøystein.

## Plugins

- [KubeVirt plugin for Headlamp](kubevirt/README.md): operations views and lifecycle actions for
  upstream KubeVirt clusters.

## Install

Published plugin releases are available on GitHub and are described with Artifact Hub metadata.

KubeVirt plugin release archive:

```text
https://github.com/stianfro/headlamp-plugins/releases/download/kubevirt-0.1.0/headlamp-plugin-kubevirt-0.1.0.tar.gz
```

KubeVirt plugin payload image for in-cluster Headlamp deployments:

```text
ghcr.io/stianfro/headlamp-plugin-kubevirt:latest
```
